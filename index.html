Professional Vehicle Gallery Website


Web Design
Web Development
HTML, CSS, JS

I developed a clean and modern vehicle gallery website focused on delivering a smooth and engaging user experience. The project showcases featured vehicles with an interactive card-based layout, allowing users to easily browse through different models. The website includes dynamic hover effects, image zoom animations, and interactive overlays that highlight key actions such as viewing more details. Each vehicle card is fully clickable and opens a responsive modal displaying additional information, creating a seamless browsing experience without navigating away from the page. The design features both light and dark themes, giving users the flexibility to choose their preferred viewing mode. A soft, minimal color palette is used in the light theme for visual comfort, while the dark theme provides a sleek and modern look suitable for extended browsing. The layout is responsive and adapts well across different screen sizes. Technologies used include HTML, CSS, and JavaScript, with a focus on smooth animations, clean structure, and user-friendly interaction. This project demonstrates my ability to create visually appealing, interactive, and user-centered web interfaces suitable for modern automotive or e-commerce platforms.
